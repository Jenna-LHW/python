# Possible Answers
Solution 1:
```
def isEven(n):
    if n%2 == 0:
        return True
    else:
        return False
    
if __name__=="__main__":
    n=int(input("Enter a number:"))
    
    if isEven(n):
        print("True")
    else:
        print("False")
```

Solution 2:
```
def isEven(n):
    return (True if n%2 == 0 else False)
    
if __name__=="__main__":
    n=int(input("Enter a number:"))
    
    print(isEven(n))
```

Most Efficient Approach
```
def isEven(n):
    return n % 2 == 0

if __name__ == "__main__":
    n = int(input("Enter a number: "))

    print(isEven(n))
```
