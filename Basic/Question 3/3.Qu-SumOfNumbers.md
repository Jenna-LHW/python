**Question 3: Given a positive integer n, find the sum of the first n natural numbers.**
Example:
```
Input: n = 3
Output: 6
Explanation: 1 + 2 + 3 = 6

Input: n = 5
Output: 15 
Explanation:  1 + 2 + 3 + 4 + 5 = 15
```
